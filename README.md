# CRM-Analytics
This project analyzes CRM data to track lead conversion, sales pipeline performance, and revenue trends.
Using SQL, data from Leads, Accounts, Opportunities, Products, and Users was cleaned, joined, and transformed into key KPIs such as conversion rate, win rate, and expected revenue.
The insights were visualized using Power BI, Tableau, and Excel through Lead and Opportunity dashboards, enabling data-driven sales decisions.
